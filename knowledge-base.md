# IT Support Knowledge Base

## How to reset your password
1. Go to company login page  
2. Click “Forgot Password”  
3. Follow email instructions  

## How to connect to Wi-Fi
- Select company Wi-Fi  
- Enter credentials  
- Restart device if connection fails  

## How to fix Zoom issues
- Restart Zoom  
- Check audio settings  
- Update application  

## How to request new software
Submit a ticket via IT portal
