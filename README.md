# IT Support Desk Implementation

## Role
IT Support Coordinator & Project Assistant

## Business Context
A growing company faced frequent IT issues including login problems, software errors, and AV setup failures. Support requests were handled informally via email and chat, leading to delays and missed tickets.

## Project Objective
To implement a structured IT Service Desk to improve incident tracking, response time, and user satisfaction.

## Tools Used
- Jira Service Management
- ITIL Framework
- Microsoft Excel
- Ticketing System
- Email & Chat Support

## Responsibilities
- Logged and prioritized IT tickets  
- Coordinated with IT engineers  
- Tracked SLA compliance  
- Updated users on issue resolution  
- Created support documentation  

## Deliverables
- Ticket Workflow  
- SLA Matrix  
- Incident Log  
- Knowledge Base  
- Performance Dashboard  

## Outcome
Ticket resolution time was reduced and employee satisfaction improved due to better tracking and communication.
