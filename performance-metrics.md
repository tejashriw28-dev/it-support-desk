# IT Support Performance Metrics

| Metric | Description |
|--------|-------------|
| Ticket Volume | Total number of tickets received |
| First Response Time | Time taken to reply to a new ticket |
| Resolution Time | Time taken to close a ticket |
| SLA Compliance | % of tickets resolved within SLA |
| User Satisfaction | Feedback score after resolution |

## Sample KPIs
- Average response time: 45 minutes  
- Average resolution time: 6 hours  
- SLA compliance: 92%  
- User satisfaction: 4.6 / 5
